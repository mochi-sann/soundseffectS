SE集マニュアル

SE集をご利用いただきありがとうございます。

このSE集に含まれている機能一覧

  • SEの再生
  • 事前に設定されたBGM・ループBGMの再生
  • サミット・Nゼミテーマソングの再生(Youtube動画)
  • N・S高校歌再生機能(YoutubeN・S高チャンネルより引用)
  • 時計機能
  • 通知音がならないタイマー
  • Ms.読み上げ(翻訳＆読み上げ機能)

各機能の詳しい解説・イチオシのポイントはこちらのマニュアルに掲載しています。（Googleスライドのサイトに移動します）
